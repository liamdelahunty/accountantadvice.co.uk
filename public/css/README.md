
# CSS File Descriptions

This file documents the purpose of each CSS file in this directory.

- `global.css`: Basic styles, variables, and utilities.
- `header.css`: Header and navigation styles.
- `hero.css`: Hero section styles.
- `buttons.css`: CTA button styles.
- `form.css`: General form styles.
- `marketo-form.css`: Specific styles for Marketo forms.
- `footer.css`: Footer styles.
- `cards.css`: Card styles.
- `rating.css`: Google rating styles.
- `marketo-form-v1.css`: Variation of Marketo form styles with 48% width for fields.
- `two-column-form.css`: Styles for a two-column form layout.
- `carousel.css`: Styles for the carousel.
- `footer-v3.css`: Variation of the footer styles.
- `modal-form.css`: Styles for the modal form.
- `landbot-button.css`: Styles for the Landbot button.
